# [u].neighbors()

## Description
Grab all the elements at the same node level (neighbors) than the `[u]` elements, excluding it. You can use a [CSS3 selector](guides/css3selectors "Learn more about CSS3 selectors"). As other Mandoo grabbing functions, it can also handle any ammount of elements and many node levels.

## Syntax
[syntax]
	u( [{String} match] )
[/syntax]

## Examples
Grab all the `<li>` elements except the first one:
[js]
	u('#nav li:first').neighbors()

	// it also can be done with a selector:
	u('#nav li:not(:first)')
[/js]

Swap visible section by anchor inside it:
[js]
	// make the section visible
	u('div.content:has(a[name="section1"])').show()

	// then hide all the others at the same level
	.neighbors('div.content').hide()
[/js]
